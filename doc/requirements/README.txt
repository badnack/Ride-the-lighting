*****READ ME*****

I dati relativi al progettino per il corso di Sistemi Intelligenti (a.a. 2011-2012) sono relativi alla modellazione di un problema di fitting allo scopo di individuare la relazione esistente tra l'irradianza solare e la luminosità di un ambiente nei diversi momenti della giornata.

Didascalia dei dati nel file datiluce_progettino.xls:

colonna A: contiene il giorno 
colonna B: contiene il mese
colonna C: contiene l'anno
colonna D: contiene l'ora
colonna E: contiene il minuto

Quindi le colonne A,B,C,D,E compongono il timestamp (data e ora) di riferimento del campione.

colonna F: contiene l'irraggiamento solare fuori dall'edificio
colonna G: contiene la luminosità rilevata nell'ambiente monitorato
colonna H: contiene l'energia media consumata per illuminare correttamente l'ambiente nell'intervallo di tempo
colonna I: contiene l'indicazione relativa al tipo di giorno (1 lavorativo, 0 festivo)
